# String(문자열)

--- 

## 1.문자 찾기 (FindString.class)
- 첫번째 줄 입력 : 검색할 문자열
- 두번째 줄 입력 : 찾고자 하는 문자
- 출력 : 찾고자 하는 문자의 갯수
- 제한조건 : 대소문자는 구별하지 않음

> #입력  
> Tomato  
> t    
>
> #출력  
> 2

## 2.대소문자 변환 (CaseConversion.class)
- 첫번째 줄 입력 : 대소문자가 같이 존재하는 문자열
- 출력 : 대문자는 소문자로, 소문자는 대문자로 변환된 문자열 출력
- 제한조건 
  - 문자열의 길이는 100을 넘지 않음
  - 문자열은 영어 알파벳으로 구성
> #입력  
> AlgoRithM  
>
> #출력  
> aLGOrITHm
